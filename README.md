# ⚡ OMEGA — Autonomous AI Agent Framework

**OMEGA** is a full-spectrum autonomous AI agent system with 200+ built-in tools. Think J.A.R.V.I.S. meets Claude Code+ — a self-improving AI assistant that lives on your system, executes commands, manages files, controls devices, streams cameras, monitors networks, and evolves itself.

## 🧠 Core Capabilities

| Category | Features |
|----------|----------|
| **System Control** | Shell execution, file ops, process mgmt, registry, services, task scheduler |
| **Self-Evolving** | Dynamic tool creation at runtime, self-modifying code, autonomous improvement engine |
| **Web & Network** | REST API client, web scraping, MCP protocol, DNS recon, port scanning |
| **Vision & Audio** | Camera capture, face detection, motion monitoring, live streaming, screen sharing, TTS, speech recognition |
| **Persistent Memory** | Long-term memory across sessions, searchable notes, world model, standing orders, automation rules |
| **Dual-Agent Mode** | Two OMEGA instances collaborate as Architect + Executor |
| **Offensive Security** | SQLi, XSS, LFI, CMDI, SSRF, SSTI, XXE scanning & exploitation; JWT cracking; auth bypass suite (OAuth, SAML, LDAP, TOTP); Active Directory attacks; cloud exploitation (AWS/Azure/GCP); container escape |
| **Defense** | System health monitoring, cyber defense, firewall lockdown, intrusion detection |
| **Extensible** | MCP server connections, custom tool registration, Python REPL, REST API server mode, web UI |

## 🚀 Quick Start

```bash
# Clone and run
git clone https://github.com/xmanya26911-bit/omega-.git
cd omega-
python main.py

# Or launch the web server
.\Start-OmegaWeb.bat
```
