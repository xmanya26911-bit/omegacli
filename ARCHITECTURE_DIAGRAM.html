OMEGA — Architecture Refactor

Event-Driven · Async-First · Plugin-Friendly · SOLID

Core Principles

  • Event-driven architecture (pub/sub)
  • Renderer abstraction (swap terminal for web)
  • Dependency injection (no hidden imports)
  • Async-first concurrency (asyncio)
  • Plugin system (auto-discover .py files)

Key Improvements

  • Zero monkey-patching (EventBus replaces `setattr`)
  • Command classes replace if/elif chains
  • 17 typed events (dataclasses with IDE support)
  • Session object replaces scattered state
  • Separate logging from terminal output

Migration Phases

  • Phase 1: core/ abstractions (no changes to existing)
  • Phase 2: renderer layer (RichRenderer + bus)
  • Phase 3: agent refactor (emit events, not prints)
  • Phase 4: CLI refactor (commands + App)
  • Phase 5: remove old code + plugin system
